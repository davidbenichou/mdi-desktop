An AngularJS Multi Document User Interface

Getting Started

About The Project

MDI-Desktop is a set of native angular directives that provide a simulated desktop windowing environment.

This project is still in an initial development phase and needs additional work.

Dependencies

The source contains a set of native angular directives. There are no dependencies on jQuery or Bootstraps Javascript. The only required dependencies are:

  • AngularJS (requires AngularJS 1.2.x, tested with 1.2.16)

  • Bootstrap CSS (tested with version 3.1.1)

  • MDI-Desktop CSS

Installation

Once the files are included in your project you just need to reference the MDI-Desktop directive in the body of your start page.

Options

Option Default Value Type Description
dataService undefined object Sets a dataservice that the window can use for validation purposes.
isDirtyFn undefined function Check for unsaved data before closing a window.
menubarHeight 0 numeric Sets the height of the menubar placeholder.
showMenubar false boolean Removes or adds menubar placeholder.
showLaunchMenu true boolean Removes or adds launch menu placeholder.
